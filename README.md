# 주식회사 동황토산 (Dong Hwang To San) 공식 소개 사이트

해외 여행객들을 대상으로 한국의 최고급 고려인삼을 홍보 및 예약 판매하고, 기업 가치와 전국 매장(본점 및 직영 지점), 자회사 브랜드를 보여주는 프리미엄 웰빙 브랜드 랜딩 웹사이트입니다.

## 📌 주요 특징
1. **다국어 완벽 지원 (Multilingual UI)**:
   - 한국어(KO), 영어(EN), 중국어 간체(ZH), 일본어(JA)의 4개 국어를 지원합니다.
   - 우측 상단 토글 메뉴를 클릭하면 페이지가 부드러운 페이드 트랜지션과 함께 실시간으로 번역됩니다.
   - **쿠키/로컬 저장소 연동**: `localStorage`를 적용하여 사용자가 어느 페이지에서 언어를 선택하더라도 서브페이지로 이동 시 해당 다국어 설정이 중단 없이 유지됩니다.
2. **동적 통화 계산기 (Dynamic Price Converter)**:
   - 선택된 국가 언어에 따라 상품 가격이 자동으로 최적의 통화 표시(₩, $, ¥, ￥)로 전환되어 외국인 여행객의 쇼핑 편의성을 극대화합니다.
3. **프리미엄 미학 (Premium Aesthetics)**:
   - 한국 전통의 기품을 드러내는 인삼 크림슨색(#800A1D) 및 골드 테마를 기본으로 채택하였습니다.
   - 자회사 **호간단 (Ho Gan Dan)** 섹션은 간 건강과 녹음의 생명력을 뜻하는 딥 포레스트 그린과 골드 테마로 자연스러운 분위기 변화를 줍니다.
   - 글래스모피즘(Glassmorphism) 네비게이션바 및 모달 카드 디자인이 적용되었습니다.
4. **상세 정보 모달 (Product Details Modal)**:
   - 상품 카드를 클릭하면 섭취 방법, 상세 성분 외에도 외국인 관광객을 위한 **국가별 세관 통관 유의사항(Customs Notice)**을 함께 표기하여 높은 구매 안심도를 제공합니다.
5. **예약 및 문의 폼 (Pre-order & Booking Form)**:
   - 본점(고양) 및 지점(부산, 제주) 방문 날짜, 인원수, 소통 가능 언어, 특별 요청 사항을 입력하여 간편하게 방문/사전 예약을 진행할 수 있습니다.
   - 제출 완료 시 깔끔하고 현대적인 애니메이션 성공 토스트(Toast) 메시지가 출력됩니다.

## 📁 폴더 구조
```
tonghwangtosan/
│
├── index.html            # 메인 홈 페이지 (요약본 및 히어로 영역)
├── ceo.html              # CEO 인사말 서브페이지
├── about.html            # 회사소개 서브페이지 (헤리티지 및 자회사 호간단 소개)

├── directions.html       # 오시는 길 서브페이지 (오시는 길 안내 및 방문 예약 폼)
│
├── styles.css            # 스타일시트 (유리막 효과, 반응형, 서브페이지 레이아웃 규칙)
├── app.js                # 다국어 유지, 가격 변환, 모달/토스트 핸들러 스크립트
├── assets/               # 이미지 자산 폴더
│   ├── hero.png          # 프리미엄 고려인삼 배경 (AI 생성)
│   ├── extract.png       # 홍삼정 제품 이미지 (AI 생성)
│   ├── roots.png         # 명품 뿌리삼 이미지 (AI 생성)
│   └── hogandan.png      # 자회사 호간단 보조제 이미지 (AI 생성)
└── README.md             # 프로젝트 안내서 (본 파일)
```

## 🚀 실행 및 테스트 방법
별도의 백엔드 설치나 패키지 빌드 없이 순수 HTML/CSS/JS로 구현되었기 때문에 정적 서버를 통해 즉시 로컬 실행이 가능합니다.

1. **로컬 서버 구동 (추천)**:
   파이썬이 설치된 환경이라면 해당 디렉토리에서 아래 명령어로 서버를 띄울 수 있습니다.
   ```bash
   python -m http.server 8000
   ```
   이후 브라우저에서 `http://localhost:8000`에 접속하여 테스트하십시오.

2. **IDE 확장 도구 사용**:
   VS Code의 **Live Server** 확장을 사용해 `index.html`에서 마우스 우클릭 후 `Open with Live Server`를 클릭하여 손쉽게 확인할 수도 있습니다.
